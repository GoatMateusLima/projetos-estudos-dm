Introducao ao HTML


O HTML (HyperText Markup Language) é a linguagem de marcação usada para criar paginas da web. Ele é usado para estruturar o conteudo na web definindo elementos(TAGS) como títulos, parágrafos, links, imagens.


HTML é composto por uma série de elementos, que são representados por tags. Por exemplo, a tag H1 é usada para definir um título principal, enquanto a tag P é usada para definir um parágrafo.

Trabalhando com formularios em HTML


Para criar formulários em HTML, usamos a tag form,que serve como um contêiner para coletar e enviar dados para o servidor (backend). O envio é feito através dos atributos method e action.


O atributo method define como os dados serão enviados: GET (dados visíveis na URL) ou POST (dados enviados no corpo da requisição). Já o atributo action indica para qual URL ou arquivo os dados serão enviados.


Dentro do formulário, podemos usar diversos elementos como input, button, select, caixas de seleção (checkbox), botões de opção (radio) e muito mais.

Estruturando seu HTML + formatações


A formatação de texto acontece quando você adapta o conteúdo para transmitir a semântica adequada. Para isso, usamos tags como mark, i, u, b, span, h1 até h6, p e muitas outras.


A estrutura do site depende da semântica e do design que você deseja passar. Podemos usar tags como div para criar contêiners, fieldset para agrupar elementos de formulário e iframe (que substituiu o antigo embed). Segundo a W3C, usar semântica no desenvolvimento web é essencial para criar páginas acessíveis, organizadas e fáceis de manter.


Por que não usamos mais embed? Porque ele foi substituído por tags específicas para mídias, como iframe, video, audio, img e svg, tornando o código mais semântico e organizado.

Trabalhando com Mídias utilizando HTML


As tags de mídia são usadas para inserir imagens, vídeos, áudios e até páginas externas dentro do site principal. Entre as principais tags temos:


img: para exibir imagens.
svg: para gráficos vetoriais.
video: para adicionar vídeos.
audio: para sons e músicas.
iframe: para incorporar páginas ou vídeos externos (como do YouTube) dentro do seu site.


Atenção: nem todos os navegadores oferecem suporte completo a todos os formatos, por isso é importante testar e, quando necessário, usar atributos como controls, autoplay e formatos compatíveis.

Criando Tabelas com HTML


Para criar tabelas usamos a tag table. Dentro dela, temos outras tags que estruturam os dados:
tr (linha da tabela).
td (célula de dados).
th (célula de cabeçalho).


Além disso, podemos agrupar partes da tabela com tags semânticas:
thead (cabeçalho da tabela, geralmente com th).
tbody (corpo da tabela, onde ficam as linhas de dados com td).
tfoot (rodapé da tabela, usado para totais ou observações).

Entendendo HTML Semântico


O que significa semântica?

Semântica é o significado ou sentido de algo dentro de um contexto.

Antes do HTML5, era comum usar apenas div para estruturar a página, sem nenhuma semântica. Com a chegada do HTML5, além da remoção de tags obsoletas, foram introduzidas tags semânticas para melhorar a interpretação do conteúdo por navegadores e motores de busca (como o Googlebot). O objetivo é separar **estrutura** (HTML) de **estilo** (CSS) e tornar a web mais acessível.


Acessibilidade:

Acessibilidade na web significa tornar o conteúdo do site acessível para todos.
Quando o navegador carrega uma página, ele interpreta o HTML e cria o DOM (Document Object Model), que é uma estrutura em árvore representando todos os elementos da página.
A partir do DOM, o navegador gera uma segunda estrutura chamada:

Árvore de Acessibilidade
, que é lida pelos leitores de tela.


O DOM sozinho não garante acessibilidade. É necessário usar **tags corretas (semânticas)** e, quando preciso, atributos ARIA (como aria-label, role) para informar ao navegador, por exemplo:

"Isso aqui é um BOTÃO"


Web Scraping:

É o processo de coletar dados de um site automaticamente. O **crawler** navega entre páginas (exemplo: Googlebot) e cria uma lista de URLs para indexação. O **scraper** extrai os dados dessas páginas para uso posterior.


Tags Estruturais:

Tags semânticas substituem o uso excessivo de div. Exemplos: header (cabeçalho), main (conteúdo principal), footer (rodapé), nav (menu de navegação), section (grupo de conteúdo relacionado), article (conteúdo independente), aside (conteúdo secundário).
Essas tags ajudam motores de busca e tecnologias assistivas a entenderem a estrutura do site.


SEO:

SEO (Search Engine Optimization) é o processo de otimizar um site para melhorar sua visibilidade nos motores de busca. Isso envolve uso de tags semânticas, palavras-chave relevantes e boa estrutura.
